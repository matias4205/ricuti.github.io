2.11- ¿Cómo varían las respuestas en el ejercicio anterior si la función, f(t), que representa la población de la colonia con el tiempo es:

f(t) = 5000 + 1000 t²      t ≥ 0
 

Tenés que haber hecho el ejercicio anterior, y haber tomado tu dosis de antibiótico. Entonces, este ejercicio te va a resultar una risa.

Nos piden la tasa de crecimiento medio entre 0 y 2 meses. te recuerdo qué es la tasa media. La tasa de variación media (voy a simbolizarla así: f'm(t1 ; t2)) es algo así como el promedio entre dos instantes respecto al tiempo Cuánto había entonces, cuánto hay ahora, con qué velocidad creció en promedio:

    
f'm(t1 ; t2) =   f(t2) — f(t1)    
t2 — t1
   
Hagamos los cálculos para el período [0 ; 2].    
f'm(0 ; 2) =   f(2) — f(0)    
2 —0
   
f'm(0 ; 2) =   5000 + 1000 2² — 5000 — 1000 0²    
2
   
  f'm(0 ; 2) = 2.000  
   
Ahora el período [0 ; 4]    
f'm(0 ; 4) =   f(4) — f(0)    
4 —0
   
f'm(0 ; 4) =   5000 + 1000 4² — 5000 — 1000 0²    
4
   
  f'm(0 ; 4) = 4.000  
   

Ahora sí, vamos a la velocidad instantánea, y nos ponemos a derivar.

f(t) = 5000 + 1000 t²

f'(t) = 2000 t

    
  f'(4) = 8.000  
   
     
   

 
DESAFIO: ¿Cuánto tiempo necesitará la población para alcanzar 14.000 individuos?
