1.7- Resolver los siguientes sistemas de ecuaciones lineales. Graficar ambas rectas e interprete el resultado hallado, o indicar si el sistema es
indeterminado.
 

a)

 x — 3y = 1 2x + 6y = 4

b)

 16y = 3x + 5 19 = 28y + 5x

c)

 5x + y — 4 = 0 10x + 2y = 4

d)

 2x = 3y — 1 4x = 6y + 2

e)

 x + y = 3 x — y = 25
   

Resolver un sistema significa encontrar los valores que satisfacen simultáneamente ambas ecuaciones. Básicamente hay cuatro métodos para alcanzar la meta: igualación, sumas y restas, sustitución y determinantes. El último se lo vamos a dejar a los expertos. Los otros tres son aplicaciones sencillas del álgebra básica. Y elegiremos entre ellas según cómo se nos presenta cada par de ecuaciones y la inspiración del momento. Vamos con la primera.

a) (Método de sustitución)

x — 3y = 1
2x + 6y = 4

De la primera despejo x, y lo que me da lo meto en la segunda.

x = 3y + 1

eso lo meto en la segunda:

2(3y + 1) + 6y = 4

6y + 2 + 6y = 4

12y = 2

y = 0,166...                 (1/6)

Con ese dato vuelvo a las ecuaciones originales para obtener x

x = 1,5

Vamos a hacer el gráfico. A esta altura me imagino que ya te das maña para esta tarea. Hay varios métodos: graficar punto y unirlos, expresar las ecuaciones en la forma y = ax + b, etcétera. Tratá al menos hacerlo una vez sin ayuda y fijate si te quedan igual que a mí.

    

Ahí las tenés, espero que coincidamos. Fijate que el punto de intersección de las rectas tiene las coordenadas que hallamos al resolver el sistema de ecuaciones. Esa es la papa. A un sistema como este se lo llama determinado.

Las paralelas no se cortan, si querés hallar el punto de intersección el álgebra te va a indicar que le estás pidiendo algo imposible: el sistema estará indeterminado.
   
Y las rectas perpendiculares son aquellas cuyas pendientes son inversas y contrarias. Entonces:    
  a)    x = 1,5   ;   y = 0,166...  
   
Los cuatro restantes te los dejo a vos para que practiques. Acá te pongo los resultados.  

 

   b)

 16y = 3x + 5 19 = 28y + 5x x = 1  ;  y = 0,5

   c)

 5x + y — 4 = 0 10x + 2y = 4 x = 0,2  ;  y = 3

   d)

 2x = 3y — 1 4x = 6y + 2 indeterminado

   e)

 x + y = 3 x — y = 25 x = 14  ;  y = —11
