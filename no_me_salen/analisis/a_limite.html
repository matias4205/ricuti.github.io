Límite
 

La noción de límite es súper sencilla y no tenés que tenerle miedo, pero es tan abstracta que no es sencilla de definir. Yo te voy a dar una definición burda, grosera, matemáticamente reprochable, vergonzosa:

El límite es una aproximación infinita.

No te olvides de que esta definición que te acabo de dar es un secreto entre vos y yo, no andes divulgándola por ahí.

Te doy un ejemplo: aproximate tanto como puedas al número 3. Tu respuesta:

2

No está mal, pero aproximate más, vos podés:

2,7

Más, no tengas miedo:

2,99

Bien, dale más:

2,999976

Perfecto... aproximate más:

2,999999999999999

Excelente, dale un cacho más:

2,9999999999999999999999999999999999

Está claro que esta aproximación es infinita, ¿no es cierto? Bien, eso es operar con límites, y este viaje mental que acabamos de realizar se puede representar simbólicamente así:
   
  lím x
x→3
   

Y se lee así: límite para x tendiendo a 3 de x. O así: límite de x para x tendiendo a 3.

A veces (pero no siempre) podemos resolver los límites. En nuestro ejemplo es muy sencillo:

    
  lím x = 3
x→3
   

Me imagino que ya estarás pensando que esto es un pavada atómica. Bueno, sí. Y entonces ¿para qué sirve?

Una de las utilidades (tiene muchas varias) es que nos permite hacer operaciones prohibidas, por ejemplo: dividir por cero (la calculadora no te lo permite, ¿no es cierto?). Mirá este ejemplo:

    
  lím 1/x
x→0
 

 

Esa operación sí se puede hacer, porque acordate que el límite es la "aproximación infinita". Ya no estás dividiendo por 0 sino que estás dividiendo por 0,001, 0,00001... etcétera. A lo largo de esa aproximación interminable la operación es lícita y tu calculadora te daría la respuesta sin titubear.

Ahora también podríamos resover ese límite... bueno, a medias.

    
  lím 1/x = ∞
x→0
   

A medias, porque ∞ (infinito) no es un número, pero al menos es una noción numérica bastante intuitiva (no me hagas que entre en ese laberinto, plis).

Tal vez la importancia fundamental del límite es que permite (a veces) resolver cocientes en los que tanto el numerador como el denominador tienden a cero, sobre todo si tanto el numerador como el denominador están relacionados entre sí por una función. Por ejemplo si quisiéramos resover este cociente: y/x, donde: y = 2x + 5, cuando x se aproxima infinitamente a cero.

Fijate que cuando xo = 0, yo = 5. (El subíndice indica que esos dos valores se corresponden entre sí). Entonces:

    
  lím x = 0
x→xo
   
Y    
  lím y = 5
y→yo
   
O lo que es lo mismo:    
  lím (x − xo) = 0
(x - xo)→0
   
Y    
  lím (y − yo) = 0
(y - yo)→0
   
Entonces:    
  lím (y − yo)  
(x - xo)→0 (x − xo)
   
Fijate que ese cociente es del tipo 0/0, ya que cuando x tiende a xo al mismo tiempo y tiende a yo. Y ya vimos que ambos límites tienden a cero. La operación 0/0 ya sabés que está prohibida, y que es imposible, y que está indeterminada, y quinientas maldiciones más... Pero si hacemos la división durante la aproximación infinita... ¡no hay nada que lo impida! Puedo seguir operando algebraicamente sin temor a represalias:    
  lím (2x + 5) − (2xo + 5)  
(x - xo)→0 (x − xo)
   
     
  lím (2x + 5 − 2xo − 5)  
(x - xo)→0 (x − xo)
   
     
  lím (2x − 2xo)  
(x - xo)→0 (x − xo)
   
     
  lím 2 (x −xo)  
(x - xo)→0 (x − xo)
   
Acordate que estamos dentro del límite, dento de la suceción infinita de aproximaciones, de modo que aunque x y xo sean muy parecidos entre sí... la resta no vale cero, entonces no estamos dividiendo por cero y podemos seguir adelante... pues, podemos simplificar el cociente:    
  lím 2
(x - xo)→0
   
El número 2 es una constante, no depende ni de y ni de x ni de nadie. De modo que:    
  lím 2 = 2
(x - xo)→0
   

Diferenciales. Esas restas que tienden a cero, se llaman diferenciales, o infinitésimos (cuando yo estudiaba los llamábamos pendejésimos), y solemos identificarlos con la letra d o ∂.

Podemos simbolizarlo así:

    
  lím (x − xo) = lím Δx = dx = ∂x  
(x - xo)→0 (x - xo)→0
   
De modo que tenés que tener en cuenta que cuando operás con un diferencial estás operando con un límite. El diferencial tiende a cero.... pero no vale cero. Podés dividir por un diferencial y no vas preso.    
Chismes importantes    
  • Los matemáticos tienen una definición (matemática) de límite. A menos que vayas a estudiar matemática no creo que te interese mucho, pero acá está:
  
  lím f(x) = L ⇔ ∀ε > 0 ∃δ > 0 / 0 < | x − c | < δ → | f(x) − L | < ε
x→c
 
 

El símbolo ∀ significa para cualquier, o para todo. El símbolo ∃ significa existe. Entonces la definición se lee así: El ímite de una función de x para x tendiendo a un valor cualquiera, c, vale L sí y sólo si para cualquier ε mayor que cero existe un δ que también es mayor que cero tal que si la diferencia en valor absoluto entre x y c es menor que δ esi implica que la diferencia en módulo entre la función de x y el límite (L) es menor que ε.

Ahora en criollo: L es el límite de una función de x cuando x tiende a c, siempre que se puede hallar para cada ocasión valor de x cercano a c tal que el valor de f(x) sea tan cercano a L como se quiera.
 
  • La operación con límites es la base del calculo infinitesimal, o cálculo diferencial, o sencillamente cálculo.
  • Uno de los límites más importantes para la física es el límite tendiendo a cero del cociente incremental de una función, también llamado: derivada.
  • Los límites tienen su propia álgebra. Tampoco debés temerle ya que es sumamente fácil e intuitiva (al menos para las cuatro operaciones básicas).
