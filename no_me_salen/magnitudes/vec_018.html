   Magnitudes y unidades
   
  

 

18- Sean los vectores:
    A = (3; 2; 1); B = (1; 0; –1); C = (0; –2; 4);
calcular y graficar cuando corresponda.

a) B x C
b) –4(B x B) – A
c) (A + B) x C
d) (A x B) • C

 

    

Estas cuentas largas y aburridas... yo sé que las sabés hacer, pero tenés que ganar práctica para no equivocarte. Si no estás suficientemente seguro tené a mano el modelo de la operación: A x B = (AyBz – AzBy  ;  AzBx – AxBz  ;  AxBy – AyBx)

B = (1; 0; –1)

C = (0; –2; 4)

B x C = (0 . 4 – (–1)(–2) ; (–1) 0 – 1 . 4 ; 1 (–2 ) – 0 . 0 )


  

OJO
los símbolos para los productos escalar y vectorial no son uniformes y dependen de la bibliografía
  B x C = (–2 ; – 4 ; –2) a)
   
b) –4(B x B) – A. Este es muy sencillo, ya que B x B nos da un verctor nulo, y si ese vector lo multiplicamos por 4 vuelve a ser el mismo nulo, o sea, que sólo hay que restarle el A.    
  –4(B x B) – A = (–3 ; –2 ; –1) b)
 

 

c) (A + B) x C. Arraquemos con la suma.

(A + B) = (3; 2; 1) + (1; 0; –1) = (4; 2; 0)

A ese lo multiplicamos vectorialmente con el C = (0; –2; 4):

        (4; 2; 0) x (0; –2; 4) = ( 2 . 4 – 0 . (–2) ; 0 . 0 – 4. 4 ; 4 . (–2) – 2. 0 )

    
  (A + B) x C = (8 ; –16 ; –8) c)
   

d) (A x B) • C. Donde: A = (3; 2; 1); B = (1; 0; –1); C = (0; –2; 4); empecemos con el producto vectorial:

(A x B) = ( 2 . (–1) – 1 . 0 ; 1 . 1 – 3 . (–1) ; 3. 0 – 2 .1)

(A x B) = (– 2; 4; – 2)    

Ahora lo multiplicamos escalarmente con C.

(A x B) • C = (– 2; 4; – 2) • (0; –2; 4)

(A x B) • C = – 2 . 0 + (–4) (–2) + (– 2) 4

    
  (A x B) • C = – 16 d)
   
En internet podés encontrar varios cacluladores de operaciones entre vectores, hacen el trabajo fácil, rápido y sin errores (que son muy comunes en cálculos extensos). Sin embargo te recomiendo hacerlos a manolpla y usar los calculadores solo para verificar el resultado... consejo de viejo.
