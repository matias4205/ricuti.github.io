   Magnitudes y unidades
   
  

 

FIS m.02- ¿Cuáles de las siguientes magnitudes son fundamentales en el sistema MKS-Internacional?
     Área, Volumen, Masa, Aceleración, Fuerza, Velocidad.
 
     

Las siglas MKS provienen de metro (patron para medir longitudes), kilogramo (patrón para medir masas) y segundo (patron para medir tiempo). Sin embargo con esos tres patrones no alcanza para medir todo lo que se puede medir.

El Sistema Internacional de Unidades, SI, también conocido como sistema métrico, adopta un set de siete unidades básicas (*):

  

 
Simb.

Magnitud

 Unidad Origen
l longitud
 metro (m) Hay metros patrones dispersos por el mundo. Actualmente se define con un trayecto de luz.
m masa kilogramo (kg) Hay kilogramos patrones dispersos por todo el mundo.
t tiempo segundo (s) Definido originalmente con movimientos astronómicos, hoy se usa un proceso atómico.
i corriente eléctrica amperio (A) Se mide la fuerza que produce la corriente entre dos conductores.
T temperatura kelvin (K) Fracción 1/273.16 de la temperatura termodinámica del punto triple del agua.
n cantidad de materia mol (mol) Se compara con la cantidad de átomos qoe hay en 12 g de carbomo 12.
Iv intensidad luminosa candela (cd) Se definen las condiciones de un experimento con una fuente de luz repoducible en cualquier parte.
   
Lo interesante de este recorte de unidades es que todas las demás unidades de cualquier cosa que se quiera medir se pueden deducir a partir de estas siete unidades básicas. Por ejemplo la unidad de fuerza, el newton, N, se deriva de las anteriores de masa, longitud y tiempo: N = kg m/s². Y así con cualquier otra magnitud que se te ocurra. De este modo, todo tiene su patrón de medida para comparar.    

(*)Tomado y adaptado de Wikipedia (es.wikipedia.org/wiki/Instrumento_de_medición).
