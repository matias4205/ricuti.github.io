3) Dada la ecuación del MRU, x = xo + v ( t – to ), despejar xo, v, t y to.

xo = x – v ( t – to )

v = ( x – xo ) / ( t – to )

t = [ ( x – xo ) / v ] + to

to = [ ( xo – x) / v ] + t
