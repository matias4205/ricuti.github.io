FIS a1.06 - Dadas dos rectas:
            r1: -2x + 3y - 3 = 0                r2: 3Cx + y + D = 0
determinar el valor de C y los posibles valores de D para que sean:
            i. coincidentes
            ii. paralelas y distintas
            iii. perpendiculares.

Puede parecer complicado... pero no lo es: hay que recordar el modelo de la funcion lineal y conocer las relaciones geométricas de las paralelas y perpendiculares.

Primero expresemos las rectas según el modelo y = a x + b

r1:      y = 0,66... x + 1
r2:      y = — 3 C x — D

Ahora el conocimiento de geometría. Dos rectas coincidentes (iguales) deben tener la misma pendiente y la misma ordenada al origen. De modo que:

0,66... = — 3 D

1 = — D

Despejamos C y D y respondemos:
 
  i.    C = 0,22... ,   D = — 1  
   
Dos paralelas tienen la misma pendiente pero distinta ordenada al origen. Luego...    
  ii.    C = 0,22... ,   D ≠ — 1  
   
Y las rectas perpendiculares son aquellas cuyas pendientes son inversas y contrarias. Entonces:    
  iii.    C = — 4,5 ,   cualquier D  
   
   

 
Desafío: Te recomiendo que grafiques estas rectas.
